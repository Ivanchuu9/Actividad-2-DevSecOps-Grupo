# Documentación del proyecto

Esta carpeta contiene la documentación complementaria de las Actividades 2 y 3.

- `Asensio_Gonzalez_Jimenez_Martin_Actividad3_Evaluacion_Seguridad.pdf`: informe final con Actividad 2 + Actividad 3.
- `justificacion_herramientas.md`: herramientas seleccionadas y motivo de uso.
- `modificaciones_app_insegura.md`: cambios introducidos para generar alertas.
- `resultados_herramientas.md`: resultados de las pruebas automáticas.
