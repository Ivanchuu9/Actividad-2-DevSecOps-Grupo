# Instrucciones de subida a GitHub - Actividad 3

1. Descomprimir el ZIP actualizado.
2. Entrar en el repositorio de GitHub de la Actividad 3.
3. Pulsar `Add file > Upload files`.
4. Arrastrar el contenido de la carpeta descomprimida.
5. Confirmar que aparecen las carpetas `actividad3-codigo-vulnerable/`, `docs/` y `evidencias/`.
6. Confirmar el commit con el mensaje:

```text
Añadir código vulnerable documentado y evidencias de la Actividad 3
```

No es necesario sustituir manualmente el código principal de la aplicación. La carpeta `actividad3-codigo-vulnerable/` queda separada para documentar el escenario inseguro usado en las pruebas.
